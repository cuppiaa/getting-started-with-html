Stage 0 Notes

Summary

This opening stage sets the stage (pun intended) for the rest of the course. It gives a high-level overview of the internet (Part 1) and then jumps into basic HTML structure and commands (Part 2).

Part 1 (a.k.a How the Web Works)

Part 1 describes the major components of how the web works:
You: Enter requests and receive information from the computer + browser
Computer+Browser: Displays information
Internet: Worlds largest network
Servers: Holds (i.e. hosts) all the files

The Internet connects to the various servers through a protocal called HTTP. This video here gives a great overview.

Part 2 (HTML = Hyper Text Markup Language)

The web is mainly made up of HTML pages. An HTML page has 4 main components
1. Text (what you see or 'content')
2. Markup (what it looks like)
3. References (images, videos)
4. Links (other web pages)

Part 2 mainly focuses on the second component of the HTML page.
A Tag directs a specific action. We learned the b (bold), br (break), a (anchor), em (emphasis), img (image), p (paragraph) tags. Some of these tags are "Inline" (e.g. b, br) and some are "Block" (e.g. p). The main difference to know now is that "Block" tags essentially put an invisible box around the content.

An Element is the tag and the content that the tag surrounds

An Attribute is a property of the Element

Finally, we reviewed how a complete HTML document looks (Doc Type, Opening HTML Tag, Header/Meta Data, Body/Content, Closing HTML tag). Good video reference here
